cuttlefish.js

Cuttlefish is a javascript implementation of the "web components" standard that will eventually be available natielly in web browsers.

Because of the limitations of current and older browsers, cuttlefish cannot implement the proposed standard "as is". Instead this library tries to respect the spirit of web component while making compromises that make it practical for a wide variety of usage scenarios.

Which Web Components Standard ?

There are many innitiatives that currently label themselve as "web components". The one covered by cuttlefish is described in these documents:

  • Introduction to Web Components
  • Introduction to Web Components Presentation by Eric Bidelman

Download and Contribute

The project is in it's early stage and does not have a "ready-made" build. Instead, we invite you to fork our GitHub repository and start playing around!

Project Status

Because the actual limitation of browsers are still to be discovered, the final feature set of cuttlefish is still unclear. Once this exploration phase ends, we will solidify the library with tests and a build process.
